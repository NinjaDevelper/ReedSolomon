reedSolomon.encodeFile(origFile, encodedFile, K, N)[source]¶

Encode file by Reed-Solomon codes with interleaving.

For interleaving, data will be considered as K x (interleaving length) matrix (interleaving length= (file size % K)+1). Then Reed Solomon coding is adapted to each column, and add parity bits to the end of column.

For example, RS(K=200,N=208) is used to 1,000,000 bytes data, which means interleaving length=5,000 so you can recover data even if you cannot download sequential 5,000*8 bytes.

Parameters:
  • origFile (str) – file name that you wan to encode.
  • encodedFile (str) – file name that you wan to output.
  • K (int) – amount of original data.
  • N (int) – total number of symbols after the erasure coding
Return 0:

is success. others for error.
